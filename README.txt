API_Web_Github

L’objectif du projet est d'utiliser l'API web ouverte de GitHub pour afficher quelques informations sur le profil d'un développeur ou d'une organisation. 
Sa documentation est disponible à cette adresse.
L'API web ouverte de GitHub est limitée à 60 appels par heure. Vous pouvez consulter votre utilisation actuelle à l'URL https://api.github.com/rate_limit.
